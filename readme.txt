Ni WooCommerce Admin Order Columns enhances the admin order page by adding columns that display crucial information about the ordered products.

== Description ==

<h4>Ni WooCommerce Admin Order Columns</h4>
<p>
Ni WooCommerce Admin Order Columns displays all sold products or items in the admin order list, offering a convenient way to quickly identify the products or items sold in each order. Eliminating the need to edit the order, this plugin provides an efficient solution to view product details directly from the order list.
</p>

<h5>Key Features of WooCommerce Admin Order Columns</h5>
<ul>
  <li>Display Order Item Product and Quantity</li>
  <li>Show Customer IP Address in Order List</li>
  <li>Show Order Product Name</li>
  <li>Show Order Product SKU</li>
  <li>Include Order Total Tax</li>
</ul>

<p>
<strong>Note</strong>: This plugin is fully compatible with both WooCommerce's new High-Performance Order Storage (HPOS) and the legacy WordPress posts storage. Whether you are utilizing HPOS or the traditional storage method, you can rely on this plugin to work seamlessly with your current setup.
</p>

== Installation ==

1. Use the WordPress plugin installer or upload files to the WordPress plugin directory.
2. Activate the plugin from the admin panel.
3. Go to `WooCommerce -> Orders` to find the new "Order Product" columns.
